Building a Chatbot: PharmacyFinder

  • Featured Skills

In this project, I built a chatbot from scratch, inspired by the instruction from the blogpost Rx Finder Chatbot using Tensorflow. The goal was to create a chatbot that tells the users the nearest pharmacy on any given day, including emergency services on the weekend. The chatbot uses a bag-of-word approach, where each sentence is classified according to its intent. The response is created using a webcrawler. The user interface was implemented using Twilio and the Telegram messenger.

Udacity Self-Driving Car Engineer: Finding Lane Lines

  • Featured Skills

In this project, I built an algorithm to detect lane lines. To detect the lane lines, several image transformations from the OpenCV library were used. The steps were: Transformation to HSV color space and color selection, grayscaling, selecting a triangular region of interest, gaussian blur, canny edge detection, hough transform and finally lane interpolation.
